## Supported statuses

<!-- snippet: DocsSamplesForStatuses.test.Theme_LYT_Mode_Text.approved.txt -->
```txt
- [ ] #task `space` Unchecked
- [x] #task `x` Checked
- [>] #task `>` Rescheduled
- [<] #task `<` Scheduled
- [!] #task `!` Important
- [-] #task `-` Cancelled
- [/] #task `/` In Progress
- [?] #task `?` Question
- [*] #task `*` Star
- [n] #task `n` Note
- [l] #task `l` Location
- [i] #task `i` Information
- [I] #task `I` Idea
- [S] #task `S` Amount
- [p] #task `p` Pro
- [c] #task `c` Con
- [b] #task `b` Bookmark
- [f] #task `f` Fire
- [k] #task `k` Key
- [w] #task `w` Win
- [u] #task `u` Up
- [d] #task `d` Down
```
<!-- endSnippet -->
