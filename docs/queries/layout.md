# Layout options
You can hide certain elements of the rendered list with the "hide" option.

The following options exist:

- `edit button`
- `backlink`
- `priority`
- `start date`
- `scheduled date`
- `due date`
- `done date`
- `recurrence rule`
- `task count`

Example:

    ```tasks
    no due date
    path includes GitHub
    hide recurrence rule
    hide task count
    hide backlink
    ```
